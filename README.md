# CSS Concepts Used

## Class Selector

```css
.head{
    color: darkblue;
}
```

## ID Selector

```css
#para{
    color: green;
}
```

## Universal Selector

```css
*{
    font-family: cursive;
}
```

## Descendant Selector

```css
#l1 a{
    background-color: beige;
}
```

## Direct Child Selector

```css
#list_item2 > li{
    color: blue;
}
```

# Output Description

- Headings are styled using classes and IDs.
- Paragraphs use different text colors.
- Links are styled differently using descendant selectors.
- List items are styled using direct child selectors.
- Horizontal lines separate sections.

# Important Notes

## Correct GitHub Link

Replace:

```html
<a href=""https://github.com"">
```

With:

```html
<a href="https://github.com">
```

because double quotes are incorrectly used.
